# PeerPrep Backend

## Setup instructions

### Setup database
- Download and run MongoDB 'mongod.exe'

### Setup environment
In terminal, run
```bash
# Windows
python -m venv env

.\env\Scripts\Activate

pip install -r requirements.txt
```
OR
```bash
python3 -m venv env

source env/bin/activate

pip install -r requirements.txt
```

### Start Backend

```bash
python launch.py
```
